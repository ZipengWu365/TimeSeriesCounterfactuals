# Feedback-driven changes in v1.8

This release focuses on distribution readiness, cross-disciplinary demos, a narrower agent onboarding path, and richer teaching assets.

## The package needed a clearer answer to why it exists above estimator packages.

v1.8 keeps the strong headline and pushes a clearer workflow-vs-estimator decision story into the README, docs home, and showcase materials.

## The repo needed stronger cross-disciplinary demos, especially beyond product and policy language.

v1.8 adds detector-downtime (physics), minimum-wage-employment (economics), and viral-attention (social science / public attention) as new flagship demos.

## The narrow agent path was good, but it could still start narrower.

v1.8 adds a starter tool profile and keeps `python -m tscfbench ...` as the default command style across docs and examples.

## The repo still lacked a lighter try-now surface and richer walkthrough notebooks.

v1.8 adds a hosted-gallery-ready docs page, a Colab-style quickstart notebook, and richer executed walkthrough notebooks with checked outputs.
